Repositório de Computação III - CCM 0218 - EP Método de Rayleigh Ritz

Nessa pasta estão incluídos:

1. splines_lineares.py
2. splines_cubicos.py
3. splines_cubicos_forca_bruta.py
4. EP3.py
5. relatório.ipynb
6. solve_SDP
7. fotos para o relatório

Os códigos dos arquivos 1 e 2 estão completamente contidos no arquivo 4, porém para facilitar a visualização e correção estamos mandando separados também.
O mesmo vale para o arquivo 3, contido no arquivo 5. Para utilizar qualquer um dos 3 primeiros códigos basta substituir as funções f, p e q pelas funções de interesse e rodar o programa. Instruções sobre como digitar os intervalos a, b e n serão dadas, bem como a decisão se o spline será linear ou cúbico para o arquivo 4.
A pasta solve_SDP se trata do pacote com nosso EP2. As fotos restantes são para o relatório.
